# Text editor

Simple editor of text

# How to use
 
 1. Choose text that you want to edit by your mouse
 2. Edit this text by choosing one of options from menu
 
 
 #Added
 1. Editing text with keyboard
    - `B - bold`
    - `I - italic`
    - `U - underlined`
    - `C - code`
    - `M - mark`
    - `Backspace - clear`
